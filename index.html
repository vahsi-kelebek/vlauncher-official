How to Install

For Linux (.AppImage)

After downloading the AppImage, you need to make it executable and then run it:

1. Make it executable:

chmod +x VLauncher-1.0.0.AppImage

2. Run it from the terminal:

./VLauncher-1.0.0.AppImage

Or, simply double-click the file after making it executable.

For Linux (.deb package)

After downloading the .deb file, install it using your system's package manager. This will also handle dependencies and add VLauncher to your applications menu automatically:

1. Install the .deb package:

sudo dpkg -i vlauncher_1.0.0_amd64.deb

2. Resolve dependencies (if prompted, this command will fix missing dependencies):

sudo apt install -f

Note: The Windows version will have its own installation instructions once available.
